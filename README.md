# PHP PixaDump

PHP Client for PixaDump

> **Note**
>
> Do not use, still in the development process. The API will change quickly.

## Installation

```shell
composer require mangodingid/pixadump-php --dev
```

## Usage

```php
pixadump('foo');
```
